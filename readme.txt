== Description ==

This plugin was made primary for [AdPress](http://wpadpress.com) users. It has some AdPress-related functionality but you can use it without having AdPress installed.

Displays useful system information for debugging purposes. It gathers information such as PHP settings, Browser used, installed plugin...

The plugin does not send these information to any third-party/service. The user is free to select/copy or download the information.

== Installation ==

There are no special requirement to run System Information

1. Upload `plugin.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does the plugin send my information to a third-party? =

No. Never.

= Where to find the System Information page? =

You can access the page under `Tools->System Information`
